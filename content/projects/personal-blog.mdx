A personal blog and portfolio built with Next.js 16, Tailwind CSS 4, and MDX. Uses GitHub as a headless CMS — each article is an MDX file in the repository, and pushing to main triggers a Vercel rebuild.

Features include tag-based filtering, reading time estimates, RSS feed generation, and an editorial/magazine aesthetic with carefully chosen typography.
